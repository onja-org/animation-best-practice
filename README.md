# animation-best-practice

### Installation

#### 1. Clone the Repository
```bash
git git@github.com:onja-org/animation-best-practice.git
cd animation-best-practice
```

### 2. Run the code 

``` 
 npm run bad
 npm run good
 npm run practices

 ``` 


#### 3. You're ready to start!
